This is an hybrid devcontainer, it will have a base-devcontainer image that has misc tools,
like `curl`, `wget`, `python`, `git` e `jdk`

Then you can choice the devcontainer cybersec category building one of the 
following DokerFile

 - `Dockerfile.web`
 - `Dockerfile.crypto`
 - `Dockerfile.software`
 - `Dockerfile.network`
 - `Dockerfile.stego`
