This repository a pwn enviroment to analyze, and exploit, an x86_64 binary. 

## vscode extensions
 - ms-python.python
 - ms-vscode.cpptools

## commnand line's tools

for the misc's tools:
 - curl
 - python3 (with uv as package manager)
 - file
 - cargo

for the softwar's sec tools:
 - ht
 - ltrace
 - gdb (with gdbserver, binutils and qemu-user-static)
 - patchelf
 - elfutils
 - pkg-config
 - pwninit
 - ropper
 - pwntools

libraries pre-installed:
- libssl-dev
- liblzma-dev
